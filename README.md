## 1. 프로젝트 소개 
다이브인은 수영클래스 통합 조회 플랫폼으로 여러 수영클래스들을 한 눈에 확인 할 수 있습니다. 
수영에 진심인 Swimmer들을 위해 수영 클래스 정보를 더 쉽게, 더 빠르게 제공합니다.

• 개발기간 : 2024.09.27 - 2024.10.06 <br><br>
• 팀 구성: BE 1명, FE 1명, 디자이너 1명, 기획자 2명 <br><br>
• [API 배포주소](https://api.dive-in.co.kr/lessons) <br><br>
• [Swagger 바로가기](https://api.dive-in.co.kr/swagger-ui/index.html)

<br>

## 2. 기술 스택
- Java 17
- Spring Boot
- JPA
- PostgreSQL
- Naver cloud platform (Server, Object Storage)
- Docker
- Github Actions
- Nginx
